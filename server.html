Web Server

This website currently runs on this linux based server. The server's hardware is a Raspberry Pi 2B, which runs headless using a lite version of a linux based operating system. Communication with the Pi is set up through secure shell and a file transfer protocol. Apache, MySQL, and PHP are all set up to run on the Raspberry Pi. Lastly, the domain is rented through Google Domains.
